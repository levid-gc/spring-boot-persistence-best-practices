SQL 异象

April 15, 2021 · 3 min read

Guan Chao

全栈工程师

SQL 异象是指：

  • 脏读（Dirty Read）
  • 不可重复读（Non-Repeatable Read）
  • 幻象读（Phantom Read）
  • 脏写（Dirty Write）
  • 读偏移（Read Skew）
  • 写偏移（Write Skew）
  • 丢失更新（Lost Update）

正如这些名称所示，这些现象表示的都是一些数据完整性异常，而导致这些问题的原因就是开发者为了压榨事务并发的性能而放宽了 SERIALIZABLE 隔离等级，使用了其他的隔离等级。

note

在选择事务隔离级别和事务并发性性能之间总是存在权衡。
